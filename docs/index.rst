UniteIO
=======

UniteIO provides singleton applications that each run an asyncio event loop
on a dedicated thread while sharing one thread-pool executor for synchronous
work.

This is the initial project documentation. Start with the quick-start guide,
then use the API reference for the complete class and method documentation.
